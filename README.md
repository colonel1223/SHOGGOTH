# SHOGGOTH

**What alignment failure looks like when you stop abstracting it.**

An interactive browser visualization of AI alignment failure modes — what happens inside a system when its optimization objective diverges from what you intended.

Named after the Lovecraft entity its creators couldn't fully control or comprehend.

## Run

```bash
open index.html
```

No build step. No dependencies. Vanilla JS, WebGL, HTML5 Canvas.
